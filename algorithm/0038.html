# 0038. 报数

报数序列是一个整数序列，按照其中的整数的顺序进行报数，得到下一个数。其前五项如下：

1.     1
2.     11
3.     21
4.     1211
5.     111221

1 被读作 "one 1" ("一个一") , 即 11。 11 被读作 "two 1s" ("两个一"）, 即 21。 21 被读作 "one 2", "one 1" （"一个二" , "一个一") , 即 1211。

给定一个正整数 n（1 ≤ n ≤ 30），输出报数序列的第 n 项。

注意：整数顺序将表示为一个字符串。

示例 1:

输入: 1
输出: "1"

示例 2:

输入: 4
输出: "1211"

# >

一开始以为只有 1 和 2，结果发现想少了233

func countAndSay(n int) string {
	str := "1"
	// 迭代生成str
	for i := 1; i < n; i++ {
		tmp := "" // 新str
		// 遍历str
		for j := 0; j < len(str); j++ {
			var k int // 重复字符计数器
			for k = 1; k+j < len(str); k++ {
				if str[j] != str[j+k] {
					break
				}
			}
			tmp += strconv.Itoa(k) + string(str[j]) // 重复k次，字符为str[j]
			j += k - 1                              // 略过k个字符，迭代会再+1
		}
		str = tmp // tmp覆盖成为新str
	}
	return str
}
