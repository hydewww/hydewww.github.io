# 0067. 二进制求和

给定两个二进制字符串，返回他们的和（用二进制表示）。

输入为非空字符串且只包含数字 1 和 0。

示例 1:

输入: a = "11", b = "1"
输出: "100"

示例 2:

输入: a = "1010", b = "1011"
输出: "10101"

# >

原先想通过 strconv.ParseUint(string, 2) 强制转换成 int 类型，做完运算后再通过字符串输出，但是有的测试用例的数超过了范围，于是老老实实逐字符相加。

为了不讨论 a 、 b 长度不同的情况，使用 getNum() 函数，当越界时直接返回 0 ，并且返回类型为 byte 简化理解。

func getNum(a string, n int) byte { // 返回byte简化理解
	if n >= 0 {
		return a[n] - '0'
	}
	return 0 // 越界时返回0
}

func addBinary(a string, b string) string {
	aLen := len(a)
	bLen := len(b)
	c := ""          // 结果
	carry := byte(0) // 进位
	for i := 0; i < aLen || i < bLen; i++ {
		switch getNum(a, aLen-i-1) + getNum(b, bLen-i-1) + carry {
		case 0:
			c = "0" + c
			carry = 0
		case 1:
			c = "1" + c
			carry = 0
		case 2:
			c = "0" + c
			carry = 1
		case 3:
			c = "1" + c
			carry = 1
		}
	}
	if carry == 1 {
		c = "1" + c
	}
	return c
}
