# 0013. 罗马数字转整数

罗马数字包含以下七种字符: I， V， X， L，C，D 和 M。

字符          数值
I             1
V             5
X             10
L             50
C             100
D             500
M             1000

例如， 罗马数字 2 写做 II ，即为两个并列的 1。12 写做 XII ，即为 X + II 。 27 写做 XXVII, 即为 XX + V + II 。

通常情况下，罗马数字中小的数字在大的数字的右边。但也存在特例，例如 4 不写做 IIII，而是 IV。数字 1 在数字 5 的左边，所表示的数等于大数 5 减小数 1 得到的数值 4 。同样地，数字 9 表示为 IX。这个特殊的规则只适用于以下六种情况：

  • I 可以放在 V (5) 和 X (10) 的左边，来表示 4 和 9。
  • X 可以放在 L (50) 和 C (100) 的左边，来表示 40 和 90。
  • C 可以放在 D (500) 和 M (1000) 的左边，来表示 400 和 900。

给定一个罗马数字，将其转换成整数。输入确保在 1 到 3999 的范围内。

示例 1:

输入: "III"
输出: 3

示例 2:

输入: "IV"
输出: 4

示例 3:

输入: "IX"
输出: 9

示例 4:

输入: "LVIII"
输出: 58
解释: L = 50, V= 5, III = 3.

示例 5:

输入: "MCMXCIV"
输出: 1994
解释: M = 1000, CM = 900, XC = 90, IV = 4.

# >

思路有点蠢233

  1. 确定位， digits 作为当前运算位的最小值，若 next 比它大则说明当前位已运算完毕，可以通过 next 确定需要运算的下一位。

  2. 确定数值，在遇到V/X前的I都是负的，后的I都是正的，若没有遇到则通过绝对值反转

    III => -1-1  = -3	=> 3
IX  => -1+5  =  4	=> 5
XII => 5+1+1 =  7	=> 7
IV  => -1+10 =  9	=> 9

具体实现如下

func rToI(ch rune) int {
	ret := 0
	switch ch {
	case 'I':
		ret = 1
	case 'V':
		ret = 5
	case 'X':
		ret = 10
	case 'L':
		ret = 50
	case 'C':
		ret = 100
	case 'D':
		ret = 500
	case 'M':
		ret = 1000
	}
	return ret
}

func addAbs(tmp int) int {
	if tmp < 0 {
		return -tmp
	}
	return tmp
}

func romanToInt(s string) int {
	result := 0    // 结果
	digits := 1000 // 当前位的最小值(1000,100,10,1)
	tmp := 0       // 当前位数值
	flag := true   // 判断是否是 4x 或 9x
	for _, ch := range s {
		next := rToI(ch)
		// 需要移位
		if digits > next {
			result += addAbs(tmp) // 已确定该位数值
			// 确定当前位
			for digits > next {
				digits /= 10
			}
			// 临时变量复位
			tmp = 0
			flag = true
		}
		// 确定数值
		if next == digits {
			if flag {
				tmp -= next
			} else {
				tmp += next
			}
		} else {
			tmp += next
			flag = false
		}
	}
	result += addAbs(tmp) // 确定最后一位的数值
	return result
}
