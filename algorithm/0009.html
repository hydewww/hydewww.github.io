# 0009. 回文数

判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

示例 1:

输入: 121
输出: true

示例 2:

输入: -121
输出: false
解释: 从左向右读, 为 -121 。 从右向左读, 为 121- 。因此它不是一个回文数。

示例 3:

输入: 10
输出: false
解释: 从右向左读, 为 01 。因此它不是一个回文数。

进阶:

你能不将整数转为字符串来解决这个问题吗？

# >

转换为字符串再逐字符比较

Golang Tip

Integer 和 string 转换使用 strconv.Itoa & strconv.Atoi

func isPalindrome(x int) bool {
	str := strconv.Itoa(x)
	for i := 0; i < len(str)/2; i++ {
		if str[i] != str[len(str)-1-i] {
			return false
		}
	}
	return true
}

# >

不转为字符串，先计算数字有几位再进行首尾比较。

func isPalindrome(x int) bool {
	if x < 0 {
		return false
	}
	tmp := x
	digits := 0 // 数字位数
	for ; tmp != 0; tmp /= 10 {
		digits++
	}
	for x != 0 {
		h := int(math.Pow10(digits - 1))
		if x/h != x%10 {
			return false
		}
		x = x % h / 10 // 去首尾
		digits -= 2
	}
	return true
}
