# 0058. 最后一个单词的长度

给定一个仅包含大小写字母和空格 ' ' 的字符串，返回其最后一个单词的长度。

如果不存在最后一个单词，请返回 0 。

**说明：**一个单词是指由字母组成，但不包含任何空格的字符串。

示例:

输入: "Hello World"
输出: 5

# >

从后向前遍历略过末尾的空格，再开始遍历计数。

func lengthOfLastWord(s string) int {
	num := 0
	for i := len(s) - 1; i >= 0; i-- {
		if s[i] != ' ' { // 略过空格
			// 计数
			for ; i >= 0; i-- {
				if s[i] != ' ' {
					num++
				} else {
					return num
				}
			}
		}
	}
	return num
}
