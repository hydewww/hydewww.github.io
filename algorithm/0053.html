# 0053. 最大子序和

给定一个整数数组 nums ，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

示例:

输入: [-2,1,-3,4,-1,2,1,-5,4],
输出: 6
解释: 连续子数组 [4,-1,2,1] 的和最大，为 6。

进阶:

如果你已经实现复杂度为 O(n) 的解法，尝试使用更为精妙的分治法求解。

# >

遍历长度为 1、2、…、 len(nums) 的所有子数组，求最大值。

func maxSubArray(nums []int) int {
	max := nums[0]
	a := make([]int, len(nums))
	for length := 0; length < len(nums); length++ { // 当前子数组长度
		for i := 0; i < len(nums)-length; i++ {
			a[i] += nums[i+length] // 当前子数组+后一个元素=长度加一的子数组
			if max < a[i] {
				max = a[i]
			}
		}
	}
	return max
}

空间复杂度 O(n) ， 时间复杂度 O(n2)
